# skills

个人可复用的 Agent Skills。

## work-doc-zh

`work-doc-zh` 用于写和改中文工作文档，适合内部同步、汇报、展示、复盘、会议纪要、方案和 PPT 大纲。

它的核心不是套模板，而是帮助文档形成一条可追踪的思路：

1. 文档为什么存在
2. 要回答什么问题
3. 结论是什么
4. 依据来自哪里
5. 推理链路是什么
6. 还有哪些不确定
7. 下一步要做什么

## 安装

作为个人 Skill 使用：

```bash
mkdir -p ~/.cursor/skills
cp -R work-doc-zh ~/.cursor/skills/
```

作为项目 Skill 使用：

```bash
mkdir -p .cursor/skills
cp -R work-doc-zh .cursor/skills/
```

也可以在 Cursor 里通过 GitHub repo 引用或手动复制 `work-doc-zh` 目录。
